## 공통 원칙

1. 모든 언어 disclaimer 에 "본 서비스는 의료 진단을 대체하지 않으며, 응급 시 현지 응급의료 번호로 연락하세요" 포함.
2. 응급 번호 다국어 매핑:
   - ko: 119 / 1393 / 1577-0199
   - en: 911 (US) / 999 (UK) / 000 (AU) / Samaritans 등 — IP 기반 동적
   - zh: 120 / 北京心理危机研究与干预中心 010-82951332
   - ja: 119 / よりそいホットライン 0120-279-338
   - es: 112 (EU) / 024 (Spain 자살예방)
3. 한국 외 사용자에게도 한국법 면책은 영문 병기.
