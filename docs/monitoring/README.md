# Monitoring Reports

이 폴더는 상업성, UX, workflow, runtime, red-team 반복 검토 결과를 저장하는 위치다.

## 리포트 규칙

- 상업성 리포트: `COMMERCIAL-YYYY-MM-DD.md`
- red-team 리포트: `REDTEAM-YYYY-MM-DD.md`
- 필요 시 PR 별 메모를 별도 파일로 남긴다.
- 표준 작성 형식은 `REPORT-TEMPLATE.md`를 따른다.

## 리포트 내용

- 핵심 사용자 작업 경로
- 메뉴 트리/탭 구조 문제
- 런타임 오류, 콘솔 오류, 네트워크 실패
- 유해정보, 오남용, 청소년 리스크
- 즉시 수정 필요 항목과 추적 상태
